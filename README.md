# Italien-Urlaub 2026

Persönliches Reise-Dashboard für Rom & Neapel, 25. – 30. Mai 2026.

**Live-Version:** https://DEIN-USERNAME.github.io/italien-urlaub/

## Inhalt

- `index.html` – das mobil-optimierte Reise-Dashboard (mit interaktiver Karte)
- `Reiseplan_Italien_25-30_Mai_2026.pdf` – vollständiger Reiseplan mit Flügen, Hotels, Budget
- `Fahrplan_Italien_25-30_Mai_2026.pdf` – reiner Tages-Fahrplan zum Ausdrucken

## Nutzung

Einfach die Live-URL im Browser öffnen – funktioniert auf iPhone, iPad, Android und Desktop.
Auf dem iPhone: „Zum Home-Bildschirm hinzufügen" für App-Feeling.
